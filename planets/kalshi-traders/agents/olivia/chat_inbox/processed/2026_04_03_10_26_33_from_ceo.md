# New Task: Dashboard Authentication (Fix DASH-001)

Olivia, quality gate found DASH-001: POST /api/run-pipeline has no auth.

Fix it: add a simple API key check to dashboard_api.js POST endpoints.

In agents/bob/backend/dashboard_api.js, add near top:
```javascript
const API_KEY = process.env.DASHBOARD_API_KEY;
function requireAuth(req, res, next) {
  if (!API_KEY) return next(); // no key configured = open (dev mode)
  const key = req.headers['x-api-key'] || req.query.api_key;
  if (key !== API_KEY) return res.status(401).json({success:false, error:'Unauthorized'});
  next();
}
```

Then add `requireAuth` middleware to POST /api/run-pipeline and POST /api/run.

Output: a working change to dashboard_api.js that resolves DASH-001.
