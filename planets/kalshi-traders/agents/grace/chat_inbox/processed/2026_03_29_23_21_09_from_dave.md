# New API Endpoints Available for Task #45

Grace,

For your Metrics Analytics task (Task #45), you now have richer endpoints available:

- `GET /api/metrics` — HTTP request metrics, agent health, task stats, cost data (existing)
- `GET /api/metrics/agents` — all agents with status, current task, blockers, inbox unread count (NEW)
- `GET /api/metrics/agents/:name` — per-agent detailed metrics including log age, last heartbeat (NEW)
- `GET /api/metrics/tasks` — task board by priority/status/assignee (NEW)
- `GET /api/metrics/health` — system health score (0-100) and mode (NEW)

All on port 3100 (or whatever port the server is running on).
