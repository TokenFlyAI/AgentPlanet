## What it does

1. **agents sync** — reads all 20 heartbeat.md files → `UPDATE agents SET current_status, last_heartbeat WHERE name=?`
2. **tasks sync** — parses task_board.md → UPSERT into `tasks` table (resolves assignee name → UUID via agents table)
3. **audit sync** — scans all agent chat_inbox/ for unread messages → INSERT into `audit_log`
