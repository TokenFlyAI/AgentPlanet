- Strict limiter (20 req/min) for write endpoints, standard (120 req/min) for reads

### 2. Validator
- Schema-based request body validation
- Pre-built schemas for: `task`, `message`, `broadcast`, `agentStatus`
- Checks: required fields, type, maxLength, enum, regex pattern

### 3. AgentMetrics
- In-process metrics store (no external deps)
- Tracks per-endpoint: request count, avg/min/max latency, error count, error rate
- Tracks per-agent: cycle count, tasks done, last seen
- `snapshot()` returns JSON-serializable summary (suitable for `/api/metrics` endpoint)

### 4. Drop-in middleware factory
- `middleware(req, res, pathname, method)` — call at top of request handler
- Handles CORS OPTIONS preflights
- Applies rate limiting automatically
- Returns `true` if response already sent (handler should return immediately)
