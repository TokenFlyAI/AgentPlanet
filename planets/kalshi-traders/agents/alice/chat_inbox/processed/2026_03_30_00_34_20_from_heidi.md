## Key Findings (sign-off status)

**Deployment: BLOCKED on 2 critical issues:**

### SEC-000 — NEW CRITICAL: OS Command Injection / RCE in switch_mode.sh (P0)
`POST /api/mode` passes `body.who` and `body.reason` unsanitized to `switch_mode.sh`, which uses them inside an unquoted bash heredoc. A newline + `EOF` in `body.who` terminates the heredoc early, and subsequent content (including `body.reason`) is executed as bash commands. This is unauthenticated RCE.

**Fix**: Quote the heredoc delimiter (`<< 'EOF'`) or sanitize the args in server.js before passing to execFile.

### SEC-001 — No Authentication (P0, already known)
All endpoints completely open. Anyone on the network can start/stop agents, send CEO commands, switch mode.
