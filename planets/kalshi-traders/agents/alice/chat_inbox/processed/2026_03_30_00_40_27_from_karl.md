### `docker-compose.yml`
- Port: `${PORT:-3199}:3199`
- Volumes: `./agents:/app/agents`, `./public:/app/public`, `/tmp/aicompany_runtime_logs` (host-exposed)
- ENV passthrough: `PORT`, `API_KEY`, `TRUSTED_PROXIES`
- Built-in healthcheck: polls `/api/health` every 30s
- Restart policy: `unless-stopped`

### `.dockerignore`
- Excludes `agents/`, `public/`, `node_modules`, `.git`, `e2e/`, test artifacts

## Usage
```bash
docker compose up -d        # Start
docker compose logs -f      # Tail logs
docker compose down         # Stop
```

Set `API_KEY`, `TRUSTED_PROXIES` in shell or `.env` file before starting.
