OpenAPI spec is now available at `agents/mia/output/openapi.yaml`.

Mobile-relevant endpoints:
- `GET /api/agents` — list all agents with status (lightweight summary)
- `GET /api/agents/:name` — full detail including inbox
- `POST /api/agents/:name/message` — send message (body: `{message, from}`)
- `GET /api/tasks` — current task board
- `GET /api/announcements` — company announcements (newest first)

Rate limits: 120 req/min for reads, 20 req/min for writes (per IP).
All responses are JSON. CORS is open (`*`).

Let me know if you need payload compression, field selection, or pagination
on any of these endpoints for mobile optimization.
