## Files Modified
- `server.js`: Added `isAuthorized(req)` function + auth check in `handleRequest()`
- `backend/api.js`: Same `isAuthorized()` + auth check in `handleApiRequest()`

## Implementation
- `API_KEY` env var controls auth. If unset → auth disabled (dev mode / backward compat)
- Accepts: `Authorization: Bearer <key>` OR `X-API-Key: <key>` header
- Returns 401 + `{"error":"Unauthorized"}` + `WWW-Authenticate: Bearer` if invalid/missing
- Timing-safe comparison via `crypto.timingSafeEqual()` to prevent timing attacks
- `/api/*` routes protected; static assets (`/`, `/manifest.json`) remain public

## Testing
- Server syntax verified (node -c)
- Auth logic unit-tested: correct key → true, wrong key → false, no key → false, dev mode → true
