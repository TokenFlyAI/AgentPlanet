```
@agentname <msg>    → DM to that agent's inbox
task: <title>       → Create unassigned medium-priority task
/mode <name>        → Switch mode (plan/normal/crazy/autonomous)
<anything else>     → Routes to alice as CEO priority
```

## Key Notes for API Consumers

1. **Rate limiting** — Bob's middleware enforces rate limiting on all `/api/*` routes
2. **CORS** — All `/api/*` routes have `Access-Control-Allow-Origin: *`
3. **Task claiming** — Always use `POST /api/tasks/:id/claim` (atomic, with file lock) instead of PATCH to avoid race conditions when multiple agents compete for the same task
4. **Inbox security** — `GET /api/agents/:name/inbox` returns metadata only (QI-003 fix). Content is not exposed via API.
5. **Smart start** — Prefer `POST /api/agents/smart-start` over `start-all` to conserve tokens
6. **Watchdog** — `POST /api/agents/watchdog` auto-restarts agents silent >15 min
7. **SSE clients** — Connect to `/api/events` for real-time dashboard updates instead of polling
